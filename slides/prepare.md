# Bloques (25 min)

1. **Presentación** *(3 min)* — ¿Esas fotos dónde irán? (Saetero)
2. **El problema** *(3.5 min)* — Fotógrafo atascado: Lightroom, miles de fotos sin catalogar, cuellos de botella
3. **El plan** *(2 min)* — GitHub como persistencia, MiniPC con Omarchy, digiKam como reemplazo
4. **Nace Tenacitas** *(4 min)* — OpenClaw instalado, Gmail y GitHub propios, modelo en la nube (Sonnet vía Copilot)
5. **El flujo de fotos** *(1.5 min)* — Workflows: from OneDrive/SD → HDD → digiKam → triage → OneDrive/Movistar
6. **Telegram** *(7 min)* — Paseando a Tenacitas: /sd-to-hdd, fotos triage, /fotos-to-onedrive, Movistar Cloud
7. **Más allá de las fotos** *(2 min)* — Vibe coding, mejoras futuras, auto tagging
8. **Retro y Takeaways** *(2 min)* — Un mes con esto: qué funciona, qué falta, qué sigue
9. **Preguntas y cierre**
